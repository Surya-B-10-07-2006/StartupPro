A comprehensive Streamlit application that generates a complete business blueprint for startup ideas using ML predictions and Google's Gemini 2.5 Flash API.

## Features

- *Success Prediction*: ML-powered probability score (0-100%)
- *Success Trajectory Chart*: Interactive 12-week projection using Plotly
- *SWOT Analysis*: AI-generated strengths, weaknesses, opportunities, and threats
- *12-Week Roadmap*: Phase-based actionable plan for the first 3 months
- *Team & Hiring Advice*: Essential roles and critical skillsets
- *Funding & Legal Advice*: Best funding path and legal foundations
- *Success Hint*: Tailored, non-obvious advice specific to your idea
- *PDF Report*: Downloadable professional report with all analysis

## Installation

1. Install dependencies:
bash
pip install -r requirements.txt


2. Set up Google AI API Key:
   - Get your API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
   - Set it as an environment variable:
   bash
   # Windows PowerShell
   $env:GOOGLE_API_KEY="your-api-key-here"
   
   # Windows CMD
   set GOOGLE_API_KEY=your-api-key-here
   
   # Linux/Mac
   export GOOGLE_API_KEY="your-api-key-here"
   

## Usage

Run the Streamlit app:
bash
streamlit run app.py


The app will open in your browser. Fill in:
1. Your startup idea description
2. Key factors (market size, competition, team experience, funding, tech complexity)
3. Click "Generate Analysis"

## Output Components

1. *Success Prediction*: Single percentage metric from ML model
2. *Strategic Line Chart*: 12-week success trajectory visualization
3. *SWOT Analysis*: Structured analysis from Gemini API
4. *12-Week Roadmap*: Phased plan with objectives and milestones
5. *Team & Hiring Advice*: Role recommendations and skillsets
6. *Funding & Legal Advice*: Funding path and legal priorities
7. *Success Hint*: One tailored piece of advice
8. *PDF Report*: Professional downloadable document

## Requirements

- Python 3.8+
- Google AI API Key
- Internet connection (for Gemini API calls)

## Notes

- The ML model uses a Random Forest classifier trained on synthetic data
- All strategic content is generated by Gemini 2.5 Flash API
- PDF generation uses ReportLab library
- Charts are interactive and created with Plotly
